# Hoomie

En mobil husholdningsapp til at styre fælles opgaver, indkøbslister og kalender med dem du bor med.

## Live Demo

[sebbbenator.github.io/hoomie](https://sebbbenator.github.io/hoomie/)

## Tech Stack

- React
- Vite
- Supabase
- React Router

## Design

Bygget og optimeret til en **393 × 852** viewport (iPhone 14, 15, 16 Pro). Åbn DevTools i din browser og sæt enheden til den opløsning for den bedste visuelle oplevelse.

## Pages

| Sti | Beskrivelse |
|---|---|
| `/` | Onboarding-flow |
| `/signin` | Log ind på din konto |
| `/signup` | Opret en ny konto |
| `/dashboard` | Startskærm med widgets |
| `/task` | Husholdningsopgaver |
| `/list` | Indkøbsliste |
| `/calendar` | Fælles husholdningskalender |
| `/profile` | Din profil |
